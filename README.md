# We Care – Hospital Analytics Dashboard
A comprehensive Power BI healthcare analytics dashboard designed to help hospitals monitor patient flow, demographics, and key performance indicators (KPIs).
# 
<a href="https://github.com/deysweta970-hub/Health-Care/commit/c7056891a184618dbeb9d36c6b91573275593715">We Care</a>

# Project Overview

This dashboard provides a unified view of hospital performance by analyzing patient admissions, demographics, doctor performance, wait time efficiency, and satisfaction levels. It enables management to take data-driven decisions to improve service quality and operational efficiency.
# Key Features
• High-Level KPIs
• 4999 Patients Treated
• 39.92 Average Age
• 35.64 Avg Waiting Time (mins)
• 4.90 Satisfaction Score
# Patient Analytics

• Admission Status: Admitted vs. Non-Admitted breakdown
• Age Group Distribution: Patients grouped from ages 0–79
• Blood Group Distribution: Counts across all major blood groups
• Gender Ratio: Male vs Female patient percentages

# Performance Insights
• Monthly Admissions Trend: Visual line chart showing patient volume across months
• Waiting Time Analysis: Percentage of patients seen within 30 minutes
• Doctor & Department Filters: Easily slice data by doctor or department
